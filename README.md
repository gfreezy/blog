# gfreezy's blog
## Initialization
```bash
brew install hugo ascidoctor
```

## Write
Posts are located in `hugo/content/posts`. Currently, asciidoctor & markdown are supported.

## Create New Post
```bash
cd hugo
hugo new posts/2020-12-18-API-返回图片格式.adoc
```

## Test
```bash
cd hugo
hugo server
```

Then access http://localhost:1313
